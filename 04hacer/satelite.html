🎯 GUÍA DE APUNTADO PASO A PASO

Paso 1: Configuración inicial del plato

  1. Orientá el plato hacia el Norte (~75º azimuth)
  2. Ajustá la elevación a 46º
  3. Rotá el LNB ~15-20º en sentido horario

Paso 2: Primera búsqueda

Usá este transpondedor para confirmar señal:

TP Frecuencia Polarización Symbol Rate Canal

Paso 3: Si no hay señal

  • Mové el plato 1-2º hacia el Este (aumentar azimuth)
  • Esperá 5 segundos entre ajustes
  • Revisá la calidad de señal en el decodificador

Paso 4: Blind scan

Una vez encontrada señal, ejecutá un blind scan para encontrar todos los canales FTA

EQUIPAMIENTO NECESARIO

Mínimo para comenzar:

  • Plato: 90cm (mínimo) - 1.5m (recomendado para Banda C)
  • LNB: Universal Ku (para FTA en Banda Ku)
  • Decodificador: FTA con blind scan
    • GT Media V8 Nova
    • Satlink WS6933
    • Azbox
    • Azamerica S2015

Para “Nova Parabólica” (Banda Ku moderna):

  • LNB: SL121 (necessario para SAT HD Regional)
  • Decodificador: Compatível com ISDB-Tb (SBTVD)
  • Plato: 60cm suficiente

COMO APUNTAR LA ANTENA

Paso a paso:

  1. Instalar el plato orientado hacia el Norte
  2. Conectar el decodificador y TV
  3. Buscar señal con el satfinder
  4. Ajustar azimuth (derecha/izquierda)
  5. Ajustar elevación (arriba/abajo)
  6. Afinar polarización del LNB
  7. Hacer blind scan para encontrar canales

Orden de búsqueda recomendada:

  1. Star One D2 (70ºW) - Más canales, señal más fuerte
  2. Star One C3 (75ºW) - Canales regionales adicionales
  3. Amazonas (61ºW) - Amazon Sat
  4. SES 6 (40.5ºW) - Canales adicionales

Tips importantes:

  • Start con D2 (70ºW) - es el más fácil desde tu ubicación
  • Desde Argentina tenés muy buena elevación (>60º) para todos
  • Un plato de 90cm alcanza para Banda Ku
  • Para Banda C necesitas 1.5m+
  • La elevación alta (62-64º) facilita mucho la recepción

FÓRMULA PARA CALCULAR AZIMUTH

Azimuth = atan2(sin(λ_sat - λ_loc), cos(lat_loc) * tan(lat_sat) - sin(lat_loc) * cos(λ_sat - λ_loc))
Elevación = 90 - acos(sqrt(1 + sin²(lat_loc) - sin²(lat_sat)) / cos(lat_loc) * cos(lat_sat))

Donde: - λ_sat = posición del satélite (negativo para Oeste) - λ_loc = longitud del observador (-56.68) - lat_loc = latitud del observador (-27.58)
